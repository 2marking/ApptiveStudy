Q)1-1 : Difference between short, int, long
A) short : 16-bit signed integer. min=-32763 Max=32767
   int   : 32-bit signed integer. min=-2147483648 Max=2147483647
   long  : 64-bit signed integer. min=--9223372036854775808 Max=9223372036854775807
   표현할수 있는 수의 크기가 다르다.
Q)1-2 : Difference between float, double
A) float : IEEE-754 32-bit real number 
   double: IEEE-754 64-bit real number
   표현할수 있는 소수점 자리수가 다르다.
Q)1-3 : Differnce between char, double
A) char : one character, primitive type-->''
   string : zero or more characters, class(encapsulates array of char)-->""

Q)2-1 : What is Encapsulation
A) Keeping fields within a class private, then providing access to them by public methods.
   필드의 클래스를 private으로 유지한체 public 으로 접근하여 마치 data 와 code를 보호할수 있는
   보호막(캡슐)을 만들어서 objects 들을 다시 쓸수 있게 만들어주는 작업.
Q)2-2 : what is Inheritance
A) letting programmers to create new classes that share some of the attributes of existing classes.
   프로그래머가 새로운 기존의 class와 일정부분을 공유하는 class 만들수 있도록 하여 같은 작업의 
   반복을 없애주는 자바의 기능.
Q)2-3 : what is Polymorphism
A) letting programmers to use same word to mean different things in different context.
   C++에서 function의 이름은 같지만 variable 이 다르면 다른 function으로 처리하는
   함수 오버라이딩과 비슷한 기능.
